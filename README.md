# Homebrew Tap for daft

Git worktree workflow tools for developers.

## Installation

```bash
brew install avihut/daft/daft
```

Or tap first:

```bash
brew tap avihut/daft
brew install daft
```

## Updating

```bash
brew update && brew upgrade daft
```
